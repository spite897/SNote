### 实例一：复制一个目录的目录结构

功能：

将一个目录及它下边的所有目录复制到另一个目录中（即创建同名目录），但不复制目录下的其他类型文件。

例子：

src 的目录结构：

src
├── a
├── b
│   ├── 1.txt
│   └── 2
│       └── 3.txt
├── c.txt
├── d
├── e f
│   └── g
│       └── 4.txt
└── g h -> e f

要构造一个 dst 目录，只包含 src 下的目录，内容如下：

dst
└── src
    ├── a
    ├── b
    │   └── 2
    ├── d
    └── e f
        └── g

思路：

首先需要先将 src 目录下的目录名筛选出来，可以用 **/*(/) 匹配。
然后用 mkdir -p 在 dst 目录中创建对应的目录。

# 参数 1：src 目录
# 参数 2：待创建的 dst 目录

#!/bin/zsh

for i ($1/**/*(/)) {
        # -p 参数是递归创建目录，这样不用考虑目录的创建顺序
        mkdir -p $2/$i
}
