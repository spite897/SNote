> ```vimscript
> set statusline+=%#warningmsg#
> set statusline+=%{SyntasticStatuslineFlag()}
> set statusline+=%*
> let g:syntastic_auto_loc_list=1
> let g:syntastic_loc_list_height=5
> ```

- pylint pyflakes pep8 mccabe       異步代碼檢查

#### virtualenv

獨立目錄包含一個pip複製 庫存放位置 編譯器軟鏈接

`常用指令`

```terminal
# 在類Unix系統中啓用virtualenv
$ virtualenv -python python3 {name}
$ source {name}/bin/activate

# 在Windows系統中啓用virtualenv
$ virtualenv -python python3 {name}
$ .\{name}\Scripts\activate

# 打包依賴
$ pip freeze > requirements.txt

# 安裝依賴
$ pip install -r requirements.txt

# 解除虛擬環境
$ deactivate
```

> - 激活後 獨立目錄路徑被前置到PATH中
> - 可執行文件中的路徑硬編碼爲絕對路徑

- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -

#### virtualenvwrapper

將虛擬環境放在同一目錄下 並提供鉤子腳本

`安裝`
```terminal
$ pip install virtualenvwrapper

# ~/.profile
export VIRTUALENVWRAPPER_PYTHON=/usr/local/bin/python3

# ~/.bash_profile
source /usr/local/bin/virtualenvwrapper.sh
```

| 常用命令            | usage    |
| -                   | -        |
| mkvirtualenv {name} |          |
| rmvirtualenv {name} |          |
| deactivate          |          |
| workon {name}       | 啓用指定 |
> http://virtualenvwrapper.readthedocs.io/en/latest/command_ref.html

> - 目錄位於 `~/.virtualenv/` `<user>\Envs` ; 通過修改 `$WORKON_HOME` 以定製
> - 提供環境名自動補全功能
