**===============================1.基本語法====================================**

```typst
= Title

statement

_italic_

+ First item
+ Second item

- This grammar is same with markdown

$Q = rho A v + C$
$Q = rho A v + "time offset"$
```

> 使用 `=` 添加標題 多個 `=` 以嵌套標題

```typst
#image("file.jpg" [, width: 70%])

you can quote like this @label

#figure(
    image("file.jpg")
    caption: [
        <mark_text>
    ],
) <label>
```

> - 使用 `#` 以進入代碼模式
> - 函數返回值可以通過 `‹label›` 接收 不進行接收則會輸出到文檔
> - **语法糖**  : func(...)[XXX] [..]   =>  func(..., [XXX], [..])

### 关于 set 关键字

为函数设置默认参数

```typst
#set par(justify: true)
-- text, page, heading, document
```

### 关于 show 关键字

```typst
#show "a_str": var => box[
    #box(image(
        "file", 
        height: 0.7em, 
    ))
    #var
]
```
