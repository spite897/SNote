```zsh

if <condition> {
}elif{}

for i (<array>){}

    (a)
    <cmd>

```

`[[..]]` 用於字符串比較;
`((..))` 用於數值比較;
`{..}` 用於運行命令;
`(..)` 用於在子shell中運行命令;

> - zsh 使用 `%` 作爲命令提示符
> - zsh 支持浮點數類型 相比於bash
> - zsh 單引號可以使用轉義字符
> - zsh 支持數組截取及負索引 `str[2,-1]`
