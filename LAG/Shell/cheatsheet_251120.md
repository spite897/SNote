### 常量

- $0    : 腳本名稱
- $1... : 腳本參數
- $$    : 腳本ID
- \$*   : "$1 $2 ...$n" - `一個字符串`
- \$@   : "$1" "$2" ... "$n" 
- $#    : 參數數量
- $?    : 上個命令退出狀態
- $HOME : 當前用戶的家目錄
- $PATH : 查找命令的路徑
- $USER : 當前登錄的用戶名
- $PWD  : 當前工作目錄
- $-, $!

- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - -

### 命令

#### :     
- 空命令

#### !!
- 上一條命令 包含參數
- - !<num> : 上n條命令
- - !<str> : 上一個以 <str> 開頭的命令

#### exec
- 執行並替換當前的 shell （即退出shell）

#### echo
- -n 禁止添加換行符
- -e 啓用轉義字符

#### read
- -p <string>     : 提示信息
- -n <num>        : 限制位數
- -t <num>        : 限制時間
- -s              : 隱藏輸入

### 運算符

| char           | desc               | e.g.                                 |
| -              | -                  | -                                    |
| 算術運算符     | --                 | --                                   |
| +-*/%          | *符號應轉義\\ \*   | \`expr $a \* $b \` or $(( $a * $b )) |
| == !=          |                    | [ $a == $b ]                         |
| 關係運算符     | --                 | --                                   |
| -eq -ne        | 相等 / 不等        | [ $a -eq $b ]                        |
| -gt -ge        | 大於 / 不小於      |                                      |
| -lt -le        | 小於 / 不大於      |                                      |
| 布爾運算符     | --                 | --                                   |
| !              | 非                 | [ ! false ]                          |
| -a -o          | 與 / 或            | [ true -o false ]                    |
| 邏輯運算符     | --                 | --                                   |
| && \|\|        |                    | [[ true && false ]]                  |
| 字符串運算符   | --                 | --                                   |
| = !=           | 相等               | [ $a = $b ]                          |
| -z -n          | 長度爲0 / 不爲零   | [ -n "$a" ]                          |
| $              | 是否不爲空         | [ $a ]                               |
| 文件測試運算符 | --                 | --                                   |
| -r -w -x       | 讀取 寫入 可執行   | [ -r file ]                          |
| -d -s -e       | 爲目錄 不爲空 存在 |                                      |
| -f -L ...      | 爲普通文件 爲鏈接  |                                      |
