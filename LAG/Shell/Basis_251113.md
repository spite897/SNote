**===============================1.數據類型====================================**

### 變量聲明 只讀變量 刪除變量 指定變量類型

```sh
# 等號兩邊不允許空格
a="asd"
# 使用變量需要加 $ 符號
echo ${a}

readonly a
# 刪除變量 不能刪除只讀變量
unset a

declare -i var_int
```

### 數組 關聯數組

```sh
var_array=(1 2 3 4 5)

declare -A var_associative_array
var_associative_array["k1"]="v1"
var_associative_array["k2"]=2
echo ${var_associative_array["k1"]}
```

### 關於常量

- $0    : 腳本名稱
- $1... : 腳本參數
- $#    : 參數數量
- $?    : 上個命令退出狀態
