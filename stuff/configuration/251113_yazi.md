提供了一个在yazi退出后同步CWD的脚本（docs中有其他Shell版本）

```zsh
function yz() {
	local tmp="$(mktemp -t "yazi-cwd.XXXXXX")" cwd
	command yazi "$@" --cwd-file="$tmp"
	IFS= read -r -d '' cwd < "$tmp"
	[ -n "$cwd" ] && [ "$cwd" != "$PWD" ] && builtin cd -- "$cwd"
	rm -f -- "$tmp"
}
```

> - 你可以通过安装 `flavor` 包来改变主题
