### 關於 Redis

redis最開始是開源項目，對於redis官方提供的服務，用戶更多選擇雲服務大廠的服務。

于是轉向閉源，但擁有資本且開源的Valkey的開發沖擊了redis的社區，造成開發人員的
流失，導致其慢性死亡。

### 關於 ElasticSearch

面對與redis相同的困境，ES擁有一個更有商業性的解決方案，即個人開源版與企業付費
版。

但雲服務壟斷級的公司 AWS 拒絕合作，在其開源版的基礎上開發了其閉源功能，並將其
開源（OpenSearch），抨擊部分開源的ES，最後ES落得與redis同樣結局。

### 關於 git : 聖人的故事

BitKeeper最初是廣泛使用的版本管理工具，但最後轉向了收費使用。

這樣背景下，Linus一氣之下，花了5天左右從零開發了git並迅速取代了前者。

此後，Linus將git毫無條件的貢獻了出去，git託管服務孵化出一個又一個獨角獸（
BitBucket Gitlab GitHub）

經濟方面，The Linux Foundation成立之初就是爲了給Linus打賞（50w/y)，此外，
RedHat也在其上市前贈與了Linus其小額股份
